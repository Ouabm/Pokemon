\doxysubsection{Detailed Description}
Gestionnaire centralisé des ressources (Singleton) 

Cette classe implémente le pattern Singleton pour fournir un accès global aux ressources du jeu (textures, polices, musiques). Elle permet \+:
\begin{DoxyItemize}
\item Le chargement et le stockage des ressources
\item L\textquotesingle{}accès sécurisé aux ressources
\item La gestion de la musique en cours
\end{DoxyItemize}

@designpattern Singleton -\/ Garantit une seule instance globale
