\doxysubsection{Detailed Description}
Représente un Pokémon avec toutes ses caractéristiques de combat. 

Cette classe encapsule les données et comportements d\textquotesingle{}un Pokémon \+:
\begin{DoxyItemize}
\item Ses attributs de base (nom, types, niveau)
\item Ses statistiques de combat (PV, attaque, défense...)
\item Ses capacités (moves)
\item Son état actuel (PV restants) 
\end{DoxyItemize}
