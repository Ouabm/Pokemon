\doxysubsection{Detailed Description}
État du menu principal du jeu, gérant la navigation entre les différents états. 

Cette classe représente l\textquotesingle{}écran principal du jeu avec les options de démarrage, d\textquotesingle{}aide et de sortie. Elle hérite de \doxylink{classGameState}{Game\+State} pour s\textquotesingle{}intégrer dans le système de gestion des états du jeu.
