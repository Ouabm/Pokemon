\doxysubsection{Detailed Description}
Représente une attaque/technique que peut utiliser un Pokémon. 

Cette classe encapsule toutes les caractéristiques d\textquotesingle{}un mouvement Pokémon \+:
\begin{DoxyItemize}
\item Son nom et son type
\item Ses caractéristiques de combat (puissance, précision)
\item Sa catégorie (physique/spécial) 
\end{DoxyItemize}
