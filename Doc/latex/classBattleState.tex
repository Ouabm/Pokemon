\doxysubsection{Detailed Description}
Classe gérant l\textquotesingle{}état de combat du jeu, héritant de \doxylink{classGameState}{Game\+State}. 

Cette classe gère toute la logique et l\textquotesingle{}affichage d\textquotesingle{}un combat Pokémon entre deux équipes. Elle hérite de la classe de base \doxylink{classGameState}{Game\+State} pour s\textquotesingle{}intégrer dans le système d\textquotesingle{}états du jeu.
