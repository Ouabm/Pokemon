\doxysubsection{Detailed Description}
Structure contenant tous les éléments visuels et logiques d\textquotesingle{}une équipe de Pokémon pendant un combat. 

Cette structure regroupe les sprites, les données des Pokémon, les éléments d\textquotesingle{}interface utilisateur et les états relatifs à une équipe pendant un combat.
