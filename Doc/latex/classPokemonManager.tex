\doxysubsection{Detailed Description}
Gestionnaire centralisé des données Pokémon (Singleton) 

Cette classe implémente le pattern Singleton pour fournir un accès global et unique à l\textquotesingle{}ensemble des données Pokémon. Elle permet \+:
\begin{DoxyItemize}
\item Le chargement centralisé des données
\item L\textquotesingle{}accès aux Pokémon par nom
\item La récupération de la liste complète
\end{DoxyItemize}

@designpattern Singleton -\/ Garantit une seule instance globale
