Detailed Description

État du menu principal du jeu, gérant la navigation entre les différents états.

Cette classe représente l'écran principal du jeu avec les options de démarrage, d'aide et de sortie. Elle hérite de GameState pour s'intégrer dans le système de gestion des états du jeu.
