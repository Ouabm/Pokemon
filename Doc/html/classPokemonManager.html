Detailed Description

Gestionnaire centralisé des données Pokémon (Singleton)

Cette classe implémente le pattern Singleton pour fournir un accès global et unique à l'ensemble des données Pokémon. Elle permet :

  • Le chargement centralisé des données
  • L'accès aux Pokémon par nom
  • La récupération de la liste complète

@designpattern Singleton - Garantit une seule instance globale
