Detailed Description

Classe gérant l'état de combat du jeu, héritant de GameState.

Cette classe gère toute la logique et l'affichage d'un combat Pokémon entre deux équipes. Elle hérite de la classe de base GameState pour s'intégrer dans le système d'états du jeu.
