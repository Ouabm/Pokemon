Detailed Description

État de fin de jeu, affichant l'écran de conclusion et les options disponibles.

Cette classe gère l'écran de fin du jeu qui s'affiche lorsqu'une partie est terminée. Elle permet au joueur de choisir entre rejouer ou quitter l'application. Hérite de GameState pour s'intégrer dans le système d'états du jeu.
