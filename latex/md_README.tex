\chapter{Simulateur de combat Pokémon}
\hypertarget{md_README}{}\label{md_README}\index{Simulateur de combat Pokémon@{Simulateur de combat Pokémon}}

 Un simulateur de combat Pokémon en C++ utilisant la bibliothèque SFML pour gérer les graphiques et les sons. Ce projet permet à deux joueurs de s\textquotesingle{}affronter en sélectionnant des Pokémon et en utilisant des attaques pour vaincre l\textquotesingle{}équipe adverse.

\begin{DoxyItemize}
\item Le joueur arrive sur un {\bfseries{menu principal}} qui propose trois options \+:
\begin{DoxyItemize}
\item {\bfseries{Start}} \+: Démarre une nouvelle partie.
\item {\bfseries{Help}} \+: Affiche des instructions sur la manière de jouer.
\item {\bfseries{Quit}} \+: Quitte le jeu.
\end{DoxyItemize}
\end{DoxyItemize}\hypertarget{md_README_autotoc_md10}{}\doxysubsection{\texorpdfstring{2. $<$strong$>$\+Sélection des Pokémon$<$/strong$>$}{2. <strong>Sélection des Pokémon</strong>}}\label{md_README_autotoc_md10}

\begin{DoxyItemize}
\item Si le joueur sélectionne {\bfseries{Start}}, il arrive dans une {\bfseries{fenêtre de sélection des Pokémon}}.
\item Le joueur peut choisir ses Pokémon \+:
\begin{DoxyItemize}
\item Le joueur {\bfseries{Bleu}} utilise les touches {\bfseries{ZQSD}} pour se déplacer et {\bfseries{Espace}} pour valider son choix.
\item Le joueur {\bfseries{Rouge}} utilise les {\bfseries{flèches directionnelles}} et {\bfseries{Entrée}} pour valider son choix.
\end{DoxyItemize}
\item Une fois les Pokémon sélectionnés, le jeu passe à la {\bfseries{fenêtre de combat}}.
\end{DoxyItemize}\hypertarget{md_README_autotoc_md11}{}\doxysubsection{\texorpdfstring{3. $<$strong$>$\+Fenêtre de Combat$<$/strong$>$}{3. <strong>Fenêtre de Combat</strong>}}\label{md_README_autotoc_md11}

\begin{DoxyItemize}
\item La fenêtre de combat affiche les Pokémon des deux équipes ainsi que leurs barres de vie.
\item En dessous, il y a un menu où chaque joueur peut choisir les {\bfseries{quatre mouvements}} du Pokémon sélectionné. Chaque équipe dispose d\textquotesingle{}une interface rectangulaire pour faire ses choix \+:
\begin{DoxyItemize}
\item Le joueur {\bfseries{Bleu}} utilise la touche {\bfseries{Tab}} pour changer de Pokémon et {\bfseries{Espace}} pour choisir un mouvement.
\item Le joueur {\bfseries{Rouge}} utilise la touche {\bfseries{P}} pour changer de Pokémon et {\bfseries{Entrée}} pour choisir un mouvement.
\end{DoxyItemize}
\item Une fois qu\textquotesingle{}un mouvement est choisi, chaque joueur valide son choix \+:
\begin{DoxyItemize}
\item {\bfseries{Espace}} pour le joueur {\bfseries{Bleu}}.
\item {\bfseries{Entrée}} pour le joueur {\bfseries{Rouge}}.
\end{DoxyItemize}
\end{DoxyItemize}\hypertarget{md_README_autotoc_md12}{}\doxysubsection{\texorpdfstring{4. $<$strong$>$\+Sélection de la Cible$<$/strong$>$}{4. <strong>Sélection de la Cible</strong>}}\label{md_README_autotoc_md12}

\begin{DoxyItemize}
\item Après avoir choisi le mouvement, une {\bfseries{cible}} apparaît à l\textquotesingle{}écran pour que le joueur choisisse le Pokémon adverse qu\textquotesingle{}il souhaite attaquer.
\begin{DoxyItemize}
\item Le joueur {\bfseries{Bleu}} utilise {\bfseries{Tab}} pour valider sa cible.
\item Le joueur {\bfseries{Rouge}} utilise {\bfseries{Entrée}} pour valider sa cible.
\end{DoxyItemize}

\begin{DoxyItemize}
\item Une fois la cible sélectionnée, le tour commence \+:
\begin{DoxyItemize}
\item Les {\bfseries{dégâts}} sont appliqués à chaque Pokémon en fonction des attaques et de leurs statistiques.
\end{DoxyItemize}
\item Le tour recommence après chaque action jusqu\textquotesingle{}à ce qu\textquotesingle{}une équipe perde ses {\bfseries{deux Pokémon}}.
\end{DoxyItemize}\hypertarget{md_README_autotoc_md14}{}\doxysubsection{\texorpdfstring{6. $<$strong$>$\+Fin de Partie$<$/strong$>$}{6. <strong>Fin de Partie</strong>}}\label{md_README_autotoc_md14}

\item La partie se termine lorsque {\bfseries{une équipe n\textquotesingle{}a plus de Pokémon en vie}}. Combat tour par tour \+: Les joueurs sélectionnent des attaques pour leurs Pokémon, et le jeu calcule les dégâts en fonction des types et des statistiques.

\item \texorpdfstring{$\ast$}{*}\+Animations \+: Des animations sont présentes pour les attaques et les changements de Pokémon.
\item \texorpdfstring{$\ast$}{*}\+Gestion des types \+: Les attaques ont des types qui influencent leur efficacité contre certains types de Pokémon.
\item \texorpdfstring{$\ast$}{*}\+Interface graphique \+: Une interface utilisateur graphique est fournie pour sélectionner les attaques et voir l\textquotesingle{}état du combat.
