Detailed Description

Classe de base abstraite pour tous les états du jeu.

Cette classe définit l'interface commune que tous les états du jeu doivent implémenter. Elle suit le pattern State pour gérer les différents états du jeu (menu, combat, etc.). Utilise un GameStateManager pour les transitions entre états.
